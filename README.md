# To-Do App  
![React](https://img.shields.io/badge/React-19-blue?logo=react) ![MUI](https://img.shields.io/badge/MUI-6.1.6-blue?logo=mui) ![Redux](https://img.shields.io/badge/Redux_Toolkit-2.2.7-red?logo=redux) ![Parcel](https://img.shields.io/badge/Parcel-2.12.0-orange)

> A **responsive, modern To-Do app** built with **React 19**, **Material UI**, **Redux Toolkit**, and **Parcel**.  
> Features: Add/Edit/Delete tasks, Search, Filter with counts, Status dropdown with colors, Snackbar feedback, LocalStorage persistence.
